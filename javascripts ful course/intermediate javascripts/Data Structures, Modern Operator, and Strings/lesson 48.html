DESTRUCTURING ARRAY

Destructuring the array in JavaScript simply means extracting multiple values from data stored in objects and arrays. The destructing assignment syntax is a JavaScript expression that makes it possible to unpack values from arrays, or properties from objects, into distinct variables.

A value can be extracted from an array and asssigned them into a different variables. Array destructing JavaScript is also used to assign and declare a variable. For destructing an array, the index value of that particular array.

What is Destructuring?

Destructuring is the method of breaking down array into simple fragment which can be used to assign a new variable. These fragments can be easily accessed and can be referenced by the variables. The destructuring of the array totally depends upon the coder and how much he wants to destroy or destruct the given array.

Syntax:
  • Array destructuring:

    var a, b;
    [a, b] = [5, 10];
    console.log(a); // 5
    console.log(b); // 10

  • Destructuring Returned Arrays:

    Those function that uses Array destructuring and returns an array makes an array functionality easy because the returned value will be a more accurate value.

    function contestants(){
    return ["contestant_1", "contestant_2", "contestant_3"]; }

    var [a, b, c] = contestants();
    console.log(a); //Output: contestant_1
    console.log(b); //Output: contestant_2
    console.log(c); // Output: contestant_3

  • Default Value:

    A default value can be assigned to a variable using array destructuring JavaScript if there is not any value or an undefined values is passed to the variable.

    Syntax:

    var x, y;
    [x = 25, y = 5] = [];
    console.log(x); //Output: 25
    console.log(y); //Output: 5
    [x = 25, y = 5] = [1, 23];
    console.log(x); //Output: 1
    console.log(y); //Output: 23

  • Ignoring Some Values:

    Some parts of an array that are not needed can be can ignored. In array destructuring JavaScript, skipping these elements of array helps concentrate on a particular part of code that are of interest. Unwanted elements can be skipped by using a trailing comma with that element.

    var a, b;
    [a, , b] = ["Person_1", "Person_2", "Person_3"];

    console.log(a); //Output: Person_1
    console.log(b); //Output: Person_2

  • Interchanging or Swapping Variables

    Array destructuring can also be used for Interchanging or swapping variables in JavaScript

    var x, y;
    [x, y] = ["Boy", "Girl"];
    [x, y] = [y, x];


    console.log(x); //Output: Girl
    console.log(y); //Output: Boy

  • Nested Array Destructuring:

    In array destructuring JavaScript, It doesn't only work for destructing an object or element of an array, but can also be nested destruction. But for doing nested destructuring, the item should also be an array so that it can be assigned to local variables.

    var numbers = [5, [2, 3, 4], 12, 14];
    var [a, [x, y, z]] = numbers;

    console.log(x); // Output: 2
    console.log(y); // Output: 3
    console.log(z); // Output: 4

  • Multiple Array Destructuring

    Array can be destructured multiple times in the same code.

    var places = ["first", "second", "third", "fourth"];
    var [x, y, , z] = [a, ...rest] = places;

    console.log(x); //Output: first
    console.log(y); //Output: fourth
    console.log(z); //Output: first
    console.log(rest); //Output: ["second", "third", "fourth"]

Conclusion:

  • Array destructuring in JavaScript is a method of breaking down an array into simplier array fragments.
  • These fragment of an array can be used to assign it to a new variable.
  • By doing array destructuring in JavaScript, many tasks can be performed such as swapping the values and nested and multiple arrays destructure.
  • The array destructuring feature comes after the ES6 update in JavaScript.
  • Array destructuring in JavaScript is almost supported in all modern browsers accept "Internet Explorer"
